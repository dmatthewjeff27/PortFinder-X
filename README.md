# PortFinder-X

High-performance multithreaded TCP port scanner with banner grabbing and JSON reporting.

## Features
- Multithreaded TCP scanning
- Banner grabbing
- Service detection
- JSON & CSV export
- CI integrated

## Usage
python main.py 192.168.1.1 --start 1 --end 1000 --threads 200
